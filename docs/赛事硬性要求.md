# 赛事硬性要求总结

> 整理自两份赛事文档，仅提取**硬性/强制性要求**，不包含建议性内容

---

## 一、参赛资格

| 要求 | 说明 |
|------|------|
| 组别 | 本科组 / 高职组（分别评审） |
| 学籍 | 仅限在读全日制本科/专科学生 |
| 队伍人数 | 1-5名选手 + 不超过2名指导老师 |
| 组队限制 | 须来自同一高校，**严禁跨校组队** |
| 每校上限 | 不超过5队 |

---

## 二、时间硬性节点

| 时间 | 要求 |
|------|------|
| 4月20日 | 报名截止，**逾期不可报名** |
| 4月25日 | 初赛作品提交截止，**逾期不可提交** |
| 5月8日-14日 | 初赛线上评审（不可更改） |
| 5月15日-6月4日 | 决赛作品提交 |
| 6月12日-13日 | 决赛线下路演 |

---

## 三、选题硬性要求

### 3.1 系统平台

| 要求 | 原文 |
|------|------|
| 操作系统 | **必须基于OpenHarmony或SwanLinkOS操作系统** |
| 南向主控 | 若采用北向+南向结合，**南向部分主控必须采用OpenHarmony操作系统** |

> ⚠️ **不可使用HarmonyOS NEXT商业版**，必须是开源鸿蒙（OpenHarmony）或SwanLinkOS

### 3.2 开发工具

| 要求 | 原文 |
|------|------|
| 开发工具 | **必须使用DevEco Studio** |
| 下载地址 | https://developer.huawei.com/consumer/cn/deveco-studio/ |

### 3.3 硬件平台

| 要求 | 原文 |
|------|------|
| 底线要求 | 硬件平台**必须能够运行OpenHarmony操作系统** |
| 纯北向替代方案 | **如纯北向设计可采用开发平台Preview功能进行作品的最终展示** |

> 即：没有硬件也能参赛，用Previewer模拟器即可

---

## 四、初赛提交物硬性要求

| 提交物 | 硬性要求 |
|--------|----------|
| 演示视频 | **一镜到底，不可剪辑**（可配音），时长不超过5分钟 |
| 汇报PPT | 必须提供（模板自备） |
| WORD技术文档 | 必须提供（格式不限，内容需涵盖项目详细、设计思路与方案、成果与应用、项目开发计划等） |
| 源代码 | 必须提供，完整可编译 |
| 提交格式 | **打包成压缩包提交** |
| 提交渠道 | 通过猿舟平台提交 |

> ⚠️ **提交截止时间后不允许更新材料**，上传前需仔细检查

---

## 五、决赛硬性要求

| 要求 | 说明 |
|------|------|
| 决赛形式 | **线下现场路演**（不可线上） |
| 汇报时长 | **10分钟**，倒计时1分钟举牌示意，结束打铃 |
| 专家提问 | **不超过5分钟** |
| 总时长 | 每队**不超过15分钟** |
| 入场要求 | 须佩戴参赛证，出示身份证+学生证，**迟到超过10分钟不得入场** |
| 汇报顺序 | 现场抽签决定 |

## 六、评分标准（硬性占比）

### 本科组

| 评分项 | 占比 | 子项 |
|--------|------|------|
| 初赛成绩 | **20%** | 提交物完整性（50分）+ 内容合理及完整性（50分） |
| 决赛路演 | **80%** | 按赛题要求设计作品及演示完成度（40分）+ 技术突破（30分）+ 创意性（20分）+ 现场表现（10分） |

### 高职组

| 评分项 | 占比 | 子项 |
|--------|------|------|
| 初赛成绩 | **20%** | 提交物完整性（50分）+ 内容合理及完整性（50分） |
| 决赛路演 | **80%** | 按赛题要求设计作品及演示完成度（40分）+ 技术突破（30分）+ 创意性（20分）+ 现场表现（10分） |

## 八、禁止事项

| 禁止 | 说明 |
|------|------|
| ❌ 跨校组队 | 每队须来自同一高校 |
| ❌ 演示视频剪辑 | 必须一镜到底 |
| ❌ 逾期补交 | 截止后不允许更新材料 |
| ❌ 冒名顶替 | 发现即取消资格 |
| ❌ 指导教师入场指导 | 决赛现场指导老师不得指导 |
| ❌ 迟到超10分钟 | 不得入场 |

## 十、硬性要求速查清单

- [ ] 队伍1-5人+不超过2名指导老师，同校

- [ ] 使用 **DevEco Studio** 开发
- [ ] 4月20日前完成报名
- [ ] 4月25日前提交：一镜到底视频 + PPT + WORD技术文档 + 源代码（压缩包）
- [ ] 视频不可剪辑，≤5分钟
- [ ] 若做南向，南向主控必须跑OpenHarmony
- [ ] 若纯北向，可用Previewer展示
- [ ] 决赛须线下路演，10分钟汇报+5分钟答辩
